This field specifies the Enterprise Data Communication Manager location.

The CES URL of the host connection will be used for obtaining an Enterprise Data license. If the CES URL is empty in the selected host connection the Compuware cloud license server will be used to obtain an Enterprise Data license using the defined Compuware customer id and site id. The Enterprise Data Communication Manager URL can be configured in Jenkins system configuration (Manage Jenkins -> Configure System -> Compuware Configurations -> Host Connections).
